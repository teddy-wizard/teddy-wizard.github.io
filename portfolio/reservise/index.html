Reservise

Simple online reservations for sports facilities

A mobile application, which allows for online reservation of courts and managing your reservations.

Implementation of the app took about one month. Of course, from time to time we are still releasing some updates and new features. The app is available on both major mobile platforms (Android and iOS) and is used in several countries in Europe.

Design and UI sketches were prepared beforehand by the Reservise’s team.

Tech wrap-up

  • a hybrid mobile application (for Android & iOS)
  • done in AngularJS and Ionic Framework
  • deployable to mobile stores as native app via Apache Cordova
  • tools used during development: Gulp, Jade, SASS, CoffeeScript
Technologies used
  • Angular
  • Ionic Framework
  • Apache Cordova
