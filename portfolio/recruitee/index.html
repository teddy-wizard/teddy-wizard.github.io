Recruitee

A beautiful online tool to help you find, screen and hire the best talent

The mobile application for Android & iOS includes almost all of the Recruitee’s web panel functionalities, like:

  • Candidate details - all candidate details and activities in one handy overview.
  • Team communication - leave internal notes for teammates or directly contact candidates.
  • Smart overview - A clear overview of all candidates and recruitment stages.
  • Powerful search - search for candidate details, resumes or test scores in the App.

The app has been available on mobile appstores for both major platforms: Android and iOS.

Design and UI sketches were prepared beforehand by the Recruitee’s team.

Tech wrap-up

  • a hybrid mobile application (for Android & iOS)
  • done in AngularJS and Ionic Framework
  • deployable to mobile stores as native app via Apache Cordova
  • tools used during development: Gulp, Jade, SASS, CoffeeScript
Technologies used
  • Angular
  • Ionic Framework
  • Apache Cordova
